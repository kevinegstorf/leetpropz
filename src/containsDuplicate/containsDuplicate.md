# containsDuplicate

## Description

Given an array of integers `nums`, determine if there are any duplicate values.

Return `true` if any number appears at least twice in the array, otherwise `false`.

### Example

```ts
Input: [1, 2, 3, 1];
Output: true;

Input: [1, 2, 3, 4];
Output: false;

Input: [1, 1, 1, 3, 3, 4, 3, 2, 4, 2];
Output: true;
```
